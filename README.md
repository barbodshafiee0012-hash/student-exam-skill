# 🎓 Student Exam Answer Skill

A Claude skill that answers exam questions exactly like a university student — no more, no less.

## What It Does

When you ask a question in exam format, Claude responds like a student sitting in an exam:

- If the question asks for **4 items** → gives exactly 4 items
- If the question asks for **2 lines** → gives exactly 2 lines
- Answers are **humanized** — natural, not robotic
- Every answer includes a **reference from Iranian university textbooks**

No unnecessary introductions. No summaries. No extra explanations. Just the answer.

## Preview

**Question:** List 3 functions of middle managers.

**Answer:**
1. Translating organizational goals into operational plans for sub-units
2. Coordinating between operational managers and top management
3. Monitoring unit performance and reporting to upper management

Reference: Rezaeian, Ali — Fundamentals of Management

## Installation

1. Download the `student-exam-answer.skill` file
2. Open [Claude.ai](https://claude.ai)
3. Go to **Settings → Capabilities → Skills**
4. Click **Install Skill** and upload the `.skill` file

> **Note:** The Skills feature is available on Pro, Team, and Enterprise plans. Make sure it's enabled in your settings.
